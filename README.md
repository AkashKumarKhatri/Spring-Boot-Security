# Spring Boot Security 

To configure the authentication mechanism of Spring Security. We’ll learn this by creating a couple of users in memory and have Spring Security authenticate and verify against them. The process you’ll learn *the* way to do authentication configuration in Spring Security and will be the useful to you irrespective of whatever your actual authentication approach is.
